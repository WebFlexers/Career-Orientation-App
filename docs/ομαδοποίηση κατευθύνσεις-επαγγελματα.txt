Βασιζόμενος στις κατευθύνσεις και τα επαγγέλματα που παραθέσατε, η ομαδοποίηση θα μπορούσε να γίνει ως εξής:

Τεχνολογία Λογισμικού και Ευφυή Συστήματα (ΤΛΕΣ)
Αυτή η κατεύθυνση επικεντρώνεται στην ανάπτυξη λογισμικού και στην εφαρμογή της τεχνητής νοημοσύνης. 
Συνεπώς, τα επαγγέλματα που θα μπορούσαν να συμπεριληφθούν εδώ είναι:

Software Developer
Web Developer
Artificial Intelligence Researcher
Data Scientist
Game Developer


Πληροφοριακά Συστήματα και Υπηρεσίες (ΠΣΥ)
Αυτή η κατεύθυνση ασχολείται με τη διαχείριση, την οργάνωση και τη στήριξη πληροφοριακών συστημάτων και υπηρεσιών. 
Τα επαγγέλματα που θα μπορούσαν να συμπεριληφθούν εδώ είναι:

Database Administrator
IT Manager
IT Consultant
Data Analyst
DevOps Engineer

Διαδικτυακά και υπολογιστικά συστήματα (ΔΥΣ)
Αυτή η κατεύθυνση ασχολείται με την κατασκευή και τη διαχείριση δικτυακών και υπολογιστικών συστημάτων. 
Τα επαγγέλματα που θα μπορούσαν να συμπεριληφθούν εδώ είναι:

Cybersecurity Analyst
Cloud Architect
Computer Hardware Engineer
Mobile App Developer
UI/UX Designer 

Σημειώνεται ότι η ομαδοποίηση αυτή είναι ενδεικτική και βασίζεται στη γενική κατανόηση και ερμηνεία των περιγραφών των κατευθύνσεων και των επαγγελμάτων. Υπάρχει περιθώριο ελαφρών μεταβολών, ανάλογα με την ακριβή περιγραφή και τα καθήκοντα του κάθε επαγγέλματος, καθώς και την ειδική έμφαση που δίνει η κάθε κατεύθυνση.
